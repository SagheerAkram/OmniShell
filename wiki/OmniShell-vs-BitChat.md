# OmniShell vs BitChat

| Feature | OmniShell | BitChat |
|:---|:---|:---|
| Protocol Count | 7 (Tor, I2P, LoRa, Sat...) | 1 (TCP/IP) |
| Architecture | Modular / Plugin-based | Monolithic |
| Encryption | AES-256-GCM + PFS | AES-256 (Static Keys) |
| Offline Capable | Yes (Mesh/LoRa) | No |
| File Transfer | Resume + Integrity Check | Basic |
| API | REST, Webhooks, WASM | None |
| License | MIT (Open) | Closed Source |
