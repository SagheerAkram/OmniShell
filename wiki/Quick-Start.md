# Quick Start Guide

## Installation

### Automatic
```bash
# Linux/macOS
curl -sL https://omnishell.dev/install.sh | bash

# Windows PowerShell
iwr -useb https://omnishell.dev/install.ps1 | iex
```

### Manual
1. Install [Rust](https://rustup.rs).
2. Clone repo: `git clone https://github.com/sagheerakram/omnishell`
3. Build: `cargo build --release`

## First Steps
1. **Initialize**: `omnishell init`
2. **Identity**: `omnishell whoami`
3. **Add Friend**: `omnishell add bob omni:XYZ...`
4. **Chat**: `omnishell msg @bob "Hello!"`
